Speech-F

HTML 5 Media Player with Search-Within-Video and Tagging feature

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Speech-F project.

Speech-F is a HTML5 Media player with Search-Within-Video and Tagging feature. Anybody can use Speech-F media player to upload video/audio files to their website using simple HTML snippets, to make the media content both searchable and taggable.

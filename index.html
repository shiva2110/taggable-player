Welcome to Speech-F project.

Speech-F is a HTML5 Media player with Search-Within-Video and Tagging feature. Speech-F media player can be used to upload video/audio files to websites using simple HTML snippets, making the media content both searchable and taggable instantly.

Authors and Contributors

Currently the sole contributor to Speech-F project is @shiva2110 .

The project is very much under development as of May 2013. However, this is a very exciting project with contributions required in technologies involving HTML5, Javascript, Jquery, Java and REST framework. If you are interested in making videos and audios of the world searchable and taggable, for instance if you are interested in changing the experience of watching a long boring lecture or a game highlights video more interesting and interactive, please contact me.

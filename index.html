Taggable-Player

HTML 5 Media Player with Search-Within-Video and Tagging feature

Download this project as a .zip file Download this project as a tar.gz file

Taggable media player can be used to upload video/audio files to websites using simple HTML5 snippets. Viewers can tag on specific points on the video and perform keyword search on existing tags.

Currently supported only in Chrome.

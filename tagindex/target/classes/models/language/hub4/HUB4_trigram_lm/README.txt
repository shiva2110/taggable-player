The language model provided is the file language_model.arpaformat.DMP,
which is a simple trigram model in binary format.
It has been built for tasks similar to broadcast news. The text used to build 
this model was taken from a veriety of permitted sources, including broadcast 
news. The vocabulary covers 64000 words, and is listed in the file called 
language_model.vocabulary.
